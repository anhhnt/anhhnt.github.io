Диагностика насоса рулевого управления с гидроусилителем

Пенящаяся или мутная жидкость рулевого управления с гидроусилителем (воздух в жидкости)

Проверки
Действия
Убедитесь в отсутствии внутренней утечки в насосе рулевого управления с гидроусилителем, вызывающей перелив.
Устраните внутреннюю утечку в насосе.
Прокачайте систему.
Проверьте, не слишком ли низок уровень жидкости.
Устраните внутреннюю утечку в насосе.
Прокачайте систему.
При низкой температуре в системе могут появляться воздушные пузырьки, если уровень жидкости слишком низкий.

Замените поликлиновый ремень привода вспомогательных агрегатов.

Скрипы в насосе рулевого управления с гидроусилителем

Проверки
Действия
Убедитесь в отсутствии воздуха в жидкости рулевого управления.
Выполните проверки, указанные ранее в разделе "Пенящаяся или мутная жидкость рулевого управления с гидроусилителем (воздух в жидкости)".

ТЕХНИЧЕСКОЕ ОБСЛУЖИВАНИЕ И РЕМОНТ

РАБОТЫ, ВЫПОЛНЯЕМЫЕ НА АВТОМОБИЛЕ




Поликлиновый ремень привода вспомогательных агрегатов

Порядок снятия

  1. Снимите бачок для охлаждающей жидкости.



  2. С помощью гаечного ключа поверните болт натяжителя по часовой стрелке, сжимая натяжитель и ослабляя натяжение поликлинового ремня привода вспомогательных агрегатов.



  3. Снимите поликлиновый ремень привода вспомогательных агрегатов.



Порядок установки

  1. С помощью гаечного ключа поверните болт натяжителя по часовой стрелке, сжимая натяжитель и ослабляя натяжение поликлинового ремня привода вспомогательных агрегатов.
  2. Удерживая гаечный ключ на болте натяжителя, проложите без натяжения поликлиновый ремень привода вспомогательных агрегатов вокруг шкивов.



  3. Наденьте ремень на натяжитель.
  4. Уберите гаечный ключ с болта натяжителя, и ремень самостоятельно натянется.



  5. Установите бачок для охлаждающей жидкости.

    6. Затянуть
    Затяните крепежные болты бачка для охлаждающей жидкости с моментом 6 Н•м (53 фунт-дюйм).




Насос в сборе

Порядок снятия

  1. Снимите поликлиновый ремень привода вспомогательных агрегатов. См. "Поликлиновой ремень привода вспомогательных агрегатов" в этом разделе.



  2. Подставьте сливной поддон, чтобы собрать жидкость рулевого управления с гидроусилителем, которая будет вытекать после того, как шланг высокого давления будет отсоединен от насоса рулевого управления.
  3. Отсоедините шланг высокого давления и подающий шланг от насоса рулевого управления с гидроусилителем.



  4. Открутите болт с правой стороны спереди насоса рулевого управления. Освободите задний правый болт и поверните в сторону кронштейн.



  5. Открутите два длинных болта насоса рулевого управления и снимите насос рулевого управления с автомобиля.



Порядок установки

  1. Установите насос рулевого управления с гидроусилителем на автомобиль и закрутите два длинных болта насоса.

    2. Затянуть
    Затяните болты насоса рулевого управления с моментом 25 Н•м (18 фунт-фут).

  2. Поверните вниз кронштейн насоса рулевого управления с гидроусилителем и вставьте болт с правой стороны спереди.

    3. Затянуть
    Затяните оба болта с правой стороны на кронштейне насоса рулевого управления с гидроусилителем с моментом 35 Н•м (26 фунт-фут).




  3. Присоедините подающий шланг к насосу рулевого управления.
  4. Присоедините фитинг шланга высокого давления к насосу рулевого управления.

    5. Затянуть
    Затяните фитинг шланга высокого давления с моментом 28 Н•м (21 фунт-фут).




  5. Установите поликлиновый ремень привода вспомогательных агрегатов. См. "Поликлиновой ремень привода вспомогательных агрегатов" в этом разделе.
  6. Заправьте бачок жидкостью для рулевого управления с гидроусилителем. См. раздел 6А, "Система рулевого управления с гидроусилителем".
  7. Прокачайте систему рулевого управления с гидроусилителем. См. раздел 6А, "Система рулевого управления с гидроусилителем".

ОБЩЕЕ ОПИСАНИЕ И ФУНКЦИОНИРОВАНИЕ СИСТЕМ

Насос рулевого управления с гидроусилителем

Насос рулевого управления с гидроусилителем представляет собой многолопастной гидравлический насос.
Поликлиновый ремень привода вспомогательных агрегатов приводит во вращение насос рулевого управления.
Насос рулевого управления обеспечивает гидравлическое давление для рулевого механизма с гидроусилителем.
Гидравлическое давление используется в рулевом механизме с гидроусилителем для облегчения управления автомобилем.
С целью освобождения пространства в моторном отсеке в системе рулевого управления с гидроусилителем используется удаленный бачок.
