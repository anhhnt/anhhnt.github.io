ОБЩЕЕ ОПИСАНИЕ

Полуоси привода колёс представляет собой блоки подвижных осей, передающих усилие от ведущего моста к передним колёсам. Каждый блок оси содержит внутренний и внешний универсальный шарнир равных угловых скоростей, соединённый с полуосью привода колеса. Внутренний шарнир полностью подвижен и может двигаться вовнутрь и наружу. Внешний шарнир также подвижен, но не может двигаться наружу и вовнутрь.
Полуоси имеют один тип наружного шарнира и один тип внутреннего шарнира. Внутренний конец каждой полуоси имеет паз, который надевается на штифт, выходящий из коробки передач.

ВЕДУЩИЙ МОСТ В СБОРЕ

Необходимое оборудование

Отделитель шарнира КМ-507-В
KM-406-A Съёмник полуоси привода колеса

Процедура снятия и установки

  1. Поднятие и подпорка автомобиля.
  2. Снять колёса. См. Раздел 2Е, Шины и колёса.
  3. Снять защиту картера двигателя. См. Раздел 9N, "Рама и нижняя часть кузова".
  4. Снять гайку полуоси привода колеса. Удалить гайку.
    5. Примечание по установке
    Момент затяжки
    Болт крепления колеса
    90 Н•м (66 фунт/фут)
    Момент затяжки
    Фиксирующая гайка
    280 Н•м (207 фунт/фут)
    • Затянуть фиксирующую гайку полуоси с моментом 180 Н•м (133 фунт/фут). Отпустить гайку и снова затянуть до 50 Н•м (37 фунт/фут). Затем ещё затянуть гайку на 60 градусов. Окончательный момент составляет 220-340 н•м (162-251 фунт/фут).
    • Долить жидкость в КПП до необходимого уровня. См. Раздел 5В. "Механическая коробка передач".



  5. Вывернуть болт и гайку нижнего шарнира.



  6. Отделить поворотный кулак от нижнего шарнира с помощью отделителя КМ-507-В.
    7. Примечание по снятию:
    Для разъединения нижнего шарового шарнира использовать только специальный инструмент. В противном случае возможно повреждение шарнира и защитного кожуха.



  7. Снять гайку поперечной рулевой тяги.
    8. Примечание по установке
    Момент затяжки
    60 Н•м (44 фунт/фут)



  8. С помощью отделителя шарнира КМ-507-В отделить конец поперечной рулевой тяги.
    9. Примечание по снятию
    Для разъединения нижнего шарового шарнира использовать только специальный инструмент. В противном случае возможно повреждение шарнира и защитного кожуха.



  9. Выдавить полуось из ступицы колеса.

    10. Важно: Подпереть незакреплённый конец полуоси привода колеса. Запрещается свободное движение полуоси от КПП после её удаления из ступицы.

    Важно: Для сбора вытекающей жидкости разместить ёмкость под коробкой передач. После удаления полуоси закрыть отверстие в коробке передач, чтобы предотвратить попадание внутрь чужеродных частиц.

  10. С помощью съёмника KM-460-A снять полуось привода колеса с коробки передач.
  11. Процедура установки идентична снятию в обратной последовательности.

РЕМОНТ АГРЕГАТА




ВНЕШНИЙ ШАРНИР

Необходимое оборудование

J-35566 Плоскогубцы для упорного кольца

Процедура демонтажа и сборки

  1. Убрать полуось из автомобиля. См. "Ведущий мост" в данном разделе.
  2. Снять большой хомут крепления защитного кожуха. Удалить хомут.
  3. Снять малый хомут крепления защитного кожуха. Удалить хомут.



  4. Удалить смазку с шарнира.
  5. Раскрыть упорное кольцо с помощью плоскогубцев J-35566 и удалить внешний шарнир с полуоси.
    6. Примечание по установке
    • В защитный кожух залить 175-195 г (6,2 - 6,9 унций) рекомендуемого смазочного материала. Защитный кожух со 175-195 г (6,2 - 6,9 унций) рекомендуемого смазочного материала установить обратно на полуось.
    • 2.0DOHC: 200-220г (7,1-7,8 унций)
    • Хомут и упорное кольцо повторно применяться не должны.



КОЖУХ ВНУТРЕННЕГО ШАРНИРА ПОЛУОСИ ПРИВОДА КОЛЕСА

Необходимое оборудование

J-35566 Плоскогубцы для упорного кольца
J-8059 Плоскогубцы для упорного кольца

Процедура демонтажа и сборки

    • Убрать полуось из автомобиля. См. "Ведущий мост" в данном разделе.
    • Снять большой хомут крепления защитного кожуха. Удалить хомут.
    • Снять малый хомут крепления защитного кожуха. Удалить хомут.



  1. Снять стопорное кольцо из корпуса внутреннего шарнира.



  2. Удалить смазку с внутреннего шарнира.
  3. С помощью плоскогубцев J 8059 снять кольцо фиксации оси.



  4. Снять внутренний шарнир и кольцо его крепления с полуоси.
  5. Снять внутреннее соединение с полуоси.
    6. Примечание по установке
    • В защитный кожух залить 195-215 г (6,9 - 7,6 унций) рекомендуемого смазочного материала. Защитный кожух со 195-215 г (6,9 - 7,6 унций) рекомендуемого смазочного материала установить обратно на полуось.
    • 2.0DOHC: 150-170г (6,9-7,6 унций)
    • Хомут и упорное кольцо повторно применяться не должны.
