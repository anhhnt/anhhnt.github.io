Прикуриватель

(показан левый привод, правый привод аналогичен)

Процедура снятия

  1. Отсоедините провод от отрицательного вывода аккумуляторной батареи.
  2. Извлечь прикуриватель из гнезда прикуривателя.
  3. Снять кожух рычага переключения передач.
  4. Отсоединить электрический разъем.



  5. Извлечь гнездо прикуривателя из кожуха прикуривателя.

    6. Внимание! Край кожуха пепельницы может стать причиной серьезной травмы.

Процедура установки

  1. Подсоединить электрический разъем.
  2. Установить гнездо прикуривателя.
  3. Установить кожух рычага переключения передач.
  4. Установить прикуриватель в гнездо прикуривателя.
  5. Установить пепельницу.
  6. Присоедините провод к отрицательному выводу аккумуляторной батареи.

Решетка переднего обогревателя

Процедура снятия

  1. Отсоединить провод от отрицательного вывода аккумуляторной батареи.
  2. Снять молдинг комбинации с панели приборов. См. "Молдинг комбинации" в этом разделе.
  3. Снять комбинацию измерителей. См. "Комбинация измерителей" в этом разделе.
  4. Снять винты и решетку переднего обогревателя.



Процедура установки

  1. Установить решетку переднего обогревателя с помощью винтов.

    2. Затянуть
    Затянуть винты решетки переднего обогревателя с усилием 3 Н•м (27 фунт-дюйм).

  2. Установить комбинацию измерителей. См. "Комбинация измерителей" в этом разделе.
  3. Установить молдинг комбинации. См. "Молдинг комбинации" в этом разделе.
  4. Присоединить провод к отрицательному выводу аккумуляторной батареи.

Панель приборов

(показан левый привод, правый привод аналогичен)

Процедура снятия

  1. Отсоедините провод от отрицательного вывода аккумуляторной батареи.
  2. Снять колпак рулевого колеса.
  3. Снять модуль подушки безопасности водителя с рулевого колеса. См. Раздел 8B, Дополнительные пневматические средства пассивной безопасности (SIR) (Для автомобилей с подушкой безопасности).
  4. Снять отделку рулевой колонки. См. Раздел 6E, Рулевое колесо и рулевая колонка.
  5. Снять рулевое колесо с рулевой колонки. См. Раздел 6E, Рулевое колесо и рулевая колонка.
  6. Снять переключатель сигналов поворота и выключатель стеклоочистителей. См. Раздел 6E, Рулевое колесо и рулевая колонка.
  7. Снять панель обивки порога передней двери. См. Раздел 9G, Облицовка салона.
  8. Поддеть и снять панели обивки передней стойки. См. Раздел 9G, Облицовка салона.
  9. Снять молдинг комбинации с панели приборов. См. "Молдинг комбинации" в этом разделе.



    10. Примечание: Во избежание повреждения панели приборов, плоскую отвертку накрыть мягкой тканью.

  10. Снять комбинацию измерителей. См. "Комбинация измерителей" в этом разделе.
  11. Снять решетку переднего обогревателя. См. "Решетка переднего обогревателя" в этом разделе.
  12. Снять сигнал включения приборов. См. "Сигнал включения приборов" в этом разделе.
  13. Снять аудиосистему с панели приборов. См. раздел 9F, «Аудиосистемы».
  14. Снять регулятор кондиционера с панели приборов. См. Раздел 7B, Ручное управление системой отопления, вентиляции и кондиционирования.



  15. Снять рычаг привода защелки капота с панели приборов. См. раздел 9R, «Передняя часть кузова».
  16. Снять узел вещевого ящика с панели приборов. См. «Вещевой ящик» в этом разделе.
  17. Извлечь модуль подушки безопасности пассажира из панели приборов. См. Раздел 8B, Дополнительные пневматические средства пассивной безопасности (SIR) (Для автомобилей с подушкой безопасности).
  18. Снять крышку переднего динамика и динамик. См. Раздел 9F, Аудиосистема.
  19. Снять переключатель управления зеркалом.
  20. Снимите напольную консоль. См. Раздел 9G, Напольная консоль






  21. Снять болты и винты с узла панели приборов.
  22. Снять панель приборов с автомобиля.



Процедура установки

  1. Установить панель приборов с помощью болтов и винты.

    2. Затянуть
    Затянуть болты панели приборов с усилием 6 Н•м (53 фунт-дюйм).

  2. Установите напольную консоль. См. Раздел 9G, Напольная консоль
  3. Установить переключатель управления зеркалом.
  4. Снять передний динамик и крышку динамика. См. Раздел 9F, Аудиосистема.
  5. Установить модуль подушки безопасности пассажира в панель приборов. См. Раздел 8B, Дополнительные пневматические средства пассивной безопасности (SIR) (Для автомобилей с подушкой безопасности).
  6. Установить узел вещевого ящика в панель приборов. См. «Вещевой ящик» в этом разделе.
  7. Установить рычаг привода защелки капота на панель приборов. См. раздел 9R, «Передняя часть кузова».
  8. Установить аудиосистему в панель приборов. См. Раздел 9F, Аудиосистема.
  9. Установить регулятор кондиционера в панель приборов. См. Раздел 7B, Ручное управление системой отопления, вентиляции и кондиционирования.
  10. Установить узел комбинации приборов. См. "Сигнал включения приборов" в этом разделе.
  11. Установить решетку переднего обогревателя. См. "Решетка переднего обогревателя" в этом разделе.
  12. Установить комбинацию измерителей. См. "Комбинация измерителей" в этом разделе.
  13. Установить молдинг комбинации на панель приборов с помощью винтов.
  14. Установить панели обивки передней стойки. См. Раздел 9G, Облицовка салона.
  15. Установить панели обивки порогов передних дверей. См. Раздел 9G, Облицовка салона.
  16. Установить переключатель сигналов поворота и выключатель стеклоочистителей. См. Раздел 6E, Рулевое колесо и рулевая колонка.
  17. Установить отделку рулевой колонки. См. Раздел 6E, Рулевое колесо и рулевая колонка.
  18. Установить рулевое колесо на рулевую колонку.
  19. Установить модуль подушки безопасности водителя в рулевое колесо. См. Раздел 8B, Дополнительные пневматические средства пассивной безопасности (SIR) (Для автомобилей с подушкой безопасности).
  20. Установить колпак рулевого колеса.
  21. Присоедините провод к отрицательному выводу аккумуляторной батареи.

Поперечина

(показан левый привод, правый привод аналогичен)

Процедура снятия

  1. Отсоедините провод от отрицательного вывода аккумуляторной батареи.
  2. Снять узел панели приборов. См. "Панель приборов" в этом разделе.
  3. Снять шланг канала обогревателя с поперечины.



  4. Снять гайки и рулевую колонку. См. Раздел 6E, Рулевое колесо и рулевая колонка.



  5. Снять канал обогревателя с верхней крышки капота.



  6. Снять болты и поперечину.



  7. Снять болты и центр поперечины.



Процедура установки

  1. Установить поперечину с помощью болтов.
  2. Установить центр поперечины с помощью болтов.

    3. Затянуть
    Затянуть болты поперечины с усилием 13 Н•м (115 фунт-дюйм)

  3. Установить канал обогревателя на верхнюю панель капота с помощью винтов.
  4. Установить рулевую колонку с помощью гаек.
  5. Установить шланг канала обогревателя с помощью винтов.
  6. Установить узел панели приборов. См. "Панель приборов" в этом разделе.
  7. Присоедините провод к отрицательному выводу аккумуляторной батареи.
