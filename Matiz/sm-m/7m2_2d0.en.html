  1. Release the parking brake.
  2. Raise the vehicle.
  3. Check the wheel bearing play by moving the top and the down of the tire in an in-and-out motion.

MAINTENANCE AND REPAIR

ON-VEHICLE SERVICE




Shock Absorber

Removal Procedure

  1. Support the rear axle with adjustable jack stands.
    • Raise the vehicle and support the rear axle assembly.



  2. Remove the shock absorber.
    • Remove the nuts.



Installation Procedure

  1. Install the shock absorber.
    • Install the shock absorber-to-rear axle nut.

    Tighten
    Tighten the bolt to 58 N•m (42 lb-ft).

    • Install the shock absorber-to-body bracket nut.

    Tighten
    Tighten the shock absorber nut to 58 N•m (42 lb-ft).

  2. Install the jack stand and lower the vehicle.



Coil Spring

Removal Procedure

  1. Support the rear axle with a jack stand.
  2. Remove the rear shock absorber. Refer to "Shock Absorber"in this section.
  3. Separate the coil spring from the rear axle by lowering the rear axle with a jack stand.

    4. Important : Lower the rear axle as long as there are no tension on the brake hose and ABS sensor wirning. So, if there is tension on them, you have to stop lowering the rear axle.




  4. Remove the coil spring
    • Remove the 2nd muffler.
    • Remove the coil spring.



Installation Procedure

  1. Install the coil spring.



  2. Install the rear shock absorber by raising the rear axle with a jack stand.
  3. Install the 2nd muffler.



Rear Axle Assembly (Including Coil Spring)

Removal Procedure

  1. Remove the rear brake drum.Refer to Section 4E, Rear Drum Brakes.
  2. Disconnect the trailing arm from the body.
    • Support the rear axle with adjustable jack stands.
    • Remove the shock absorber-to-rear axle nut.
    • Remove the shock absorber-to-body bracket nut.



    • Remove the trailing arm-to-body bracket bolt.

    Caution : When disconnecting the rear axle and the trailing arm, coil spring tension may cause the adjustable jack stand to slip from the rear axle. This may result in personal injury. Carry out the operation with co-worker.




  3. Disconnect the brake pipe.
  4. Remove the ABS sensor.



  5. Remove the parking brake lever.



  6. Remove the parking brake cable from the rear brake parking hook. Refer to Section 4G, Parking Brake.



  7. Remove the rear axle from the vehicle.
    • Remove the coil spring by lowering the jack stand slowly.
    • Lower the support jack and remove the rear axle.



  8. Remove the rear bumper stopper, rear spring upper seat and rear spring lower seat.



Installation Procedure

  1. Install in the reverse order of removal.

    2. Important : Make sure the spring diameter prior to installing the spring. Upper spring diameter is smaller than lower spring diameter.

    Caution : Support the jack stand to the rear axle securely until installing the trailing arm bolts.

  2. Raise the rear axle with a jack stand.
  3. Install the rear bumper stoper, rear spring upper seat and rear spring lower seat.



  4. Install the trailing arm bolt and nut.

    5. Tighten
    Tighten the trailing arm bolt to 80 N•m (59 lb-ft).




  5. Position the coil spring and install the shock short.



  6. Install the parking brake lever.



  7. Connecting the parking brake cable with the hook.



  8. Connect the brake pipe and install the ABS sensor.
  9. Perform the bleeding of the brake system.



Trailing Arm Bushing

Removal Procedure

  1. Remove the rear axle. Refer to "Rear Axle" in this section.
  2. Remove the trailing arm bushing using a press.



Installation Procedure

  1. Insert the trailing arm bushing to the trailing arm using a press.
  2. Install the rear axle.

GENERAL DESCRIPTION AND SYSTEM OPERATION

Rear Suspension

The rear suspension consists of an axle with two coil springs, two shock absorbers, two upper spring seats, and two bump stoppers. The axle structure maintains the relationship of the wheels to the body.
Each coil spring is retained between a seat in the underbody and a seat welded to the top of the rear axle.
