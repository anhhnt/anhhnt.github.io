MAINTENANCE AND REPAIR

ON-VEHICLE SERVICE




Headlamps

Removal Procedure

  1. Support the hood suitably.
  2. Disconnect the negative battery cable.



  3. Remove the headlamp.
    • Remove the bolts and washer.
    • Disconnect the electrical connectors.
    • Remove the headlamp.

Installation Procedure

  1. Install the headlamp with the bolts.

    2. Tighten
    Tighten the headlamp assembly bolts to 4 N•m (35 lb-in).

  2. Connect the headlamp connector and turn signal lamp connector.
  3. Connect the negative battery cable.



Turn Signal Lamps

Removal Procedure

  1. Disconnect the negative battery cable.
  2. Remove the headlamp. Refer to "Headlamp" in this section.
  3. Remove the turn signal lamp bulb.



Installation Procedure

  1. Install the turn signal lamp bulb.
  2. Install the headlamp. Refer to "Headlamp" in this section.
  3. Connect the negative battery cable.



Side Turn Signal Lamps

Removal Procedure

  1. Disconnect the negative battery cable.
  2. Slide the side turn signal lamp rearward.
  3. Remove the lamp.
  4. Disconnect the electrical connector.



Installation Procedure

  1. Connect the electrical connector.
  2. Install the side turn signal lamp.
  3. Connect the negative battery cable.



Rear Combination Lamps

Removal Procedure

  1. Disconnect the negative battery cable.
  2. Open the tailgate.
  3. Remove the rear combination lamps.
    • Remove the screws.
    • Remove the taillamp.



Installation Procedure

  1. Connect the electrical connector.

    2. Tighten
    Tighten the rear combination lamp assembly screws to 4 N•m (35 lb-in).

  2. Install rear combination lamps with the screws.
  3. Connect the negative battery cable.



Front Fog Lamp

Removal Procedure

  1. Disconnect the negative battery cable.
  2. Raise and suitably support the vehicle.
  3. Remove the front wheels.
  4. Remove the front wheel house splash shield.
    • Remove the clip and bolts.
    a. Bolt and clips.
    • Remove the wheel house splash shield.
    b. Splash shield.



  5. Remove the front fog lamp.
    • Remove the screws.
    • Disconnect the electrical connector.
    • Remove the front fog lamp.



Installation Procedure

  1. Install the front fog lamp with the screws.

    2. Tighten
    Tighten the front fog lamp screws to 4 N•m (35 lb-in).

  2. Connect the front fog lamp connector.
  3. Install the front wheelhouse splash shields with the clips and the bolts.
  4. Install the front wheels.
  5. Lower the vehicle.
  6. Connect the negative battery cable.



Rear Fog/Backup Lamp(s)

Removal Procedure

  1. Disconnect the negative battery cable.
  2. Remove the rear fog/backup lamp assembly.
  3. Remove the bulb(s).



Installation Procedure

  1. Install the bulb(s).
  2. Install the rear fog/backup lamp assembly.
  3. Connect the electrical connector.



Center High-Mounted Stoplamp (CHMSL)

Removal Procedure

  1. Disconnect the negative battery cable.
  2. Open the tailgate.
  3. Remove the screws and the CHMSL.



  4. Remove the CHMSL cover screws.
  5. Remove the CHMSL bulb.



Installation Procedure

  1. Install the CHMSL bulb.
  2. Install the CHMSL cover with the screws.
  3. Install the CHMSL with the screws.

    4. Tighten
    Tighten the CHMSL cover screws to 2 N•m (18 lb-in).

  4. Connect the negative battery cable.



License Plate Lamps

Removal Procedure

  1. Disconnect the negative battery cable.
  2. Remove the screws.
  3. Remove the license plate lamp from the tailgate handle.
  4. Remove the bulb.



Installation Procedure

  1. Install a new bulb.
  2. Connect the electrical connector.

    3. Notice : Dissimilar metals in direct contact with each other may corrode rapidly. Make sure to use the correct fasteners to prevent premature corrosion.

  3. Install the lamp assembly with the screws.
  4. Connect the negative battery cable.



Door Jamb Switch

Removal Procedure

  1. Disconnect the negative battery cable.
  2. Remove the screw and the door jamb switch.
  3. Disconnect the electrical connector.



Installation Procedure

  1. Connect the electrical connector.

    2. Notice : Dissimilar metals in direct contact with each other may corrode rapidly. Make sure to use the correct fasteners to prevent premature corrosion.

  2. Install the door jamb switch with the screw.

    3. Tighten
    Tighten the door jamb switch screw to 2 N•m (18 lb-in).




Interior Courtesy Lamp

Removal Procedure

  1. Disconnect the negative battery cable.
  2. Pry off the interior coutesy lamp lens by inserting a screwdriver into the recess along the edge of the lens.
  3. Remove the screws and the interior coutesy lamp housing from the headliner.
  4. Disconnect the electrical connector.
  5. Remove the bulb.



Installation Procedure

  1. Install a new bulb.
  2. Connect the electrical connector.

    3. Notice : Dissimilar metals in direct contact with each other may corrode rapidly. Make sure to use the correct fastener to preuent premature corrosion.

  3. Install the interior coutesy lamp housing to the headliner with the screws.
  4. Press the courtesy lamp lens onto the housing.
  5. Connect the negative battery cable.



Daytime Running Lamp Module

Removal Procedure

  1. Disconnect the negative battery cable.
  2. Disconnect the electrical connector.
  3. Remove the screws and the daytime rounning lamp module.



Installation Procedure

  1. Install the daytime running lamp module with the screws.

    2. Tighten
    Tighten the daytime running lamp module screws to 3 N•m (27 lb-in).

  2. Connect the electrical connector.
  3. Connect the negative battery cable.



Luggage Room Lamp (Only Europe Region)

Removal Procedure

  1. Disconnect the negative battery cable.
  2. Gently pry the luggage compartment lamp off by inserting a flathead screwdriver into the recess on the edge of the lamp.
  3. Disconnect the electrical connector.
  4. Remove the bulb.



Installation Procedure

  1. Install a new bulb.
  2. Connect the electrical connector.
  3. Insert the luggage compartment lamp into the recess and press the lens into place.
  4. connect the negative battery cable.



Battery Saver

Removal Procedure

  1. Disconnect the negative battery cable.
  2. Remove the battery saver.
  3. Disconnect the electrical connector.



Installation Procedure

  1. Connect the electrical connector.
  2. Install the battery saver.
  3. Connect the negative battery cable.
