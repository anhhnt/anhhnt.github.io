MAINTENANCE AND REPAIR

ON-VEHICLE SERVICE




Steering Wheel

(Left-Hand Drive Shown, Right-Hand Drive Similar)

Tools Required

DW310-010, Steering Wheel Puller

Removal Procedure

  1. Disconnect the negative battery cable.
  2. Remove the SIR module. Refer to Section 8B, Supplemental Inflatable Restraints.(For vehicle with SIR)
  3. Remove the steering wheel.
    • Remove the steering wheel horn cap (For vehicle without SIR).



    • Remove the steering wheel nut.
    • In order to install the steering wheel correctly, match-mark the steering column shaft to the steering wheel.
    • Remove the steering wheel using the steering wheel puller DW310-010.

    Important : Be careful not to hit the top of steering column shaft with hammer during repair work.




Installation Procedure

  1. Install the SIR module. Refer to Section 8B, Supplemental Inflatable Restraints.

    2. Important : Align the match marks on the steering wheel and the steering column shaft.

  2. Install the steering wheel.
    • Install the steering wheel nut to the steering column shaft.

    Tighten
    TightenTighten the steering wheel nut to 38 N•m (28 lb-in).

  3. Connect the negative battery cable.

Installation Procedure

    Tighten the steering column cover panel screws to 2.5 N•m (22 lb-in).




Signal & Light Switch and Lever/ Wiper Switch and Lever

(Left-Hand Drive Shown, Right-Hand Drive Similar)

Removal Procedure

  1. Disconnect the negative battery cable.
  2. Remove the steering column cover. Refer to "Steering Column Cover"in this section.
  3. Remove the signal & light switch.
    • Disconnect the signal & light switch by pushing the locking from the switch housing (1).
    • Disconnect the connector (2).



  4. Remove the wiper switch.
    • Disconnect the wiper switch by pushing the locking from the switch housing (1).
    • Disconnect the connector (2).



Installation Procedure

  1. Install the wiper switch and connect the connector.



  2. Install the signal & light switch and connect the connector.
  3. Install the steering column cover. Refer to "Steering Column Cover"in this section.
  4. Connect the negative battery cable.



Steering Lock Set

(Left-Hand Drive Shown, Right-Hand Drive Similar)

Removal Procedure

  1. On vehicle equipped with immobilizer, remove the steering wheel. Refer to "Steering Column Cover"in this section.
  2. Remove the steering column cover. Refer to "Steering Column Cover"in this section.
  3. On vehicle equipped with immobilizer, remove immobilizer.
    • Disconnect the connector (1).
    • Remove the screw (2).
    • Remove the immobilizer (3).



  4. With the key in the ignition turn to the position designated ACC, remove steering lock set by pressing down the detent spring and pulling the steering lock set out of the steering lock housing.



Installation Procedure

    Important : To install the steering lock set the ignition position should be designated ACC.

  1. Install the steering lock set.



  2. On vehicle equipped with immobilizer install the immobilizer.
    • Install the screw (1).
    • Connect the connector (2).
  3. Install the steering column cover. Refer to "Steering Column Cover"in this section.
  4. On vehicle equipped with immobilizer, install the steering wheel. Refer to "Steering Wheel" in this section.



Steering Column Assembly

(Left-Hand Drive Shown, Right-Hand Drive Similar)

Removal Procedure

  1. Remove the steering wheel. Refer to "Steering Wheel" in this section.

    2. Caution : Disconnect the negative battery cable and let the vehicle sit for 1 minute to deactivate the airbag (For vehicle with SIR)

  2. Remove the signal & light switch and wiper switch. Refer to "Signal & Light Switch and Lever /Wiper Switch and Lever"in this section.



  3. Remove the steering lock set. Refer to "Steering Lock Set"in this section.
  4. Remove the steering column assembly.
    • Remove the intermediate shaft upper pinch bolt (1).
    • Remove the steering column lower nuts (2).



    • Remove the steering column upper nuts (3).



Installation Procedure

  1. Install the steering column assembly.
    • Install the steering column upper nuts (a).

    Tighten
    Tighten the upper nut to 17.5 N•m (13 lb-ft).




    • Install the steering column lower nuts (b).

    Tighten
    Tighten the lower nut to 17.5 N•m (13 lb-ft).




    • Install the intermediate shaft lower pinch bolt (c).

    Tighten
    Tighten the pinch bolt to 25 N•m (18 lb-ft).

  2. Install the steering lock set. Refer to "Steering Lock Set"in this section.
  3. Install the Signal & Light switch and wiper switch. Refer to "Signal & Light Switch and Lever/Wiper Switch and Lever"in this section.
  4. Install the steering wheel. Refer to "Steering Wheel"in this section.

GENERAL DESCRIPTION AND SYSTEM OPERATION

Steering Wheel and Column

In addition to the steering function, the steering column provides safety and security.

Caution : To ensure energy-absorbing action, it is important to use only the specified screws, bolts, and nuts, tightened to the specified torque.

The energy-absorbing column is designed to compress in a front-end collision to lessen the chance of driver injury.
The ignition switch and the lock are mounted on the column, allowing the ignition and steering operations to be locked to inhibit theft of the car.
The column levers trigger the turn signals, the headlight beams, and the windshield washer and wipers.

Notice : Apply a thin coat of lithium grease to all friction points when reassembling to ensure proper operation.

The column may be easily disassembled and assembled.
