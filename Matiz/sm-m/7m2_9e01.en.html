Cigar Lighter

(Left-Hand Drive Shown, Right-Hand Drive Similar)

Removal Procedure

  1. Disconnect the negative battery cable.
  2. Remove the cigar lighter from the cigar lighter socket.
  3. Remove the gearshift lever boot.
  4. Disconnect the electrical connector.



  5. Remove the cigar lighter socket from the cigar lighter housing.

    6. Caution : Ashtray housing edge could cause serious injury.

Installation Procedure

  1. Connect the electrical connector.
  2. Install the cigar lighter socket.
  3. Install the gearshift lever boot.
  4. Install the cigar lighter in the cigar lighter socket.
  5. Install the ashtray.
  6. Connect the negative battery cable.



Cluster Molding

(Left-Hand Drive Shown, Right-Hand Drive Similar)

Removal Procedure

  1. Disconnect the battery negative cable.
  2. Remove the cluster molding from the instrument panel.

Installation Procedure

  2. Install the cluster molding to the instrument cluster trim panel.
  3. Connect the battery negative cable.

Instrument Panel

(Left-Hand Drive Shown, Right-Hand Drive Similar)

Removal Procedure

  1. Disconnect the negative battery cable.
  2. Remove the steering wheel cap.
  3. Remove the driver's airbag module from the steering wheel. Refer to Section 8B, Supplemental Inflatable Restraints (SIR) (For the vehicle equipped with the airbag.)
  4. Remove the steering column trim cover. Refer to Section 6E, Steering Wheel and Column.
  5. Remove the steering wheel from the steering column. Refer to Section 6E, Steering Wheel and Column.
  6. Remove the signal lamp switch and the wiper switch. Refer to Section 6E, Steering Wheel and Column.
  7. Remove the front rocker trim panel. Refer to Section 9G, Interior Trim.
  8. Pry off the A-pillar trim panels. Refer to Section 9G, Interior Trim.
  9. Remove the cluster molding from the instrument panel. Refer to "Cluster Molding"in this section.



    10. Notice : To prevent the instrument panel damage, cover the flathead screwdriver with a soft cloth.

  10. Remove the meter cluster. Refer to "Meter Cluster"in this section.
  11. Remove the front defroster grille. Refer to "Front Defroster Grille"in this section.
  12. Remove the instrument telltail. Refer to "Instrument Telltail"in this section.
  13. Remove the audio system from the instrument panel. Refer to Section 9F, Audio Systems.
  14. Remove the A/C controller from the instrument. Refer to Section 7B, Manual Control Heating, Ventilation, and Air Conditioning System.



  15. Remove the hood latch release lever from the instrument panel. Refer to Section 9R, Body Front End.
  16. Remove the glove box assembly from the instrument panel. Refer to "Glove Box"in this section.
  17. Remove the passenger's airbag module from the instrument panel. Refer to Section 8B, Supplemental Inflatable Restraints (SIR) (For vehicle equipped with the airbag).
  18. Remove the front speaker cover and speaker. Refer to Section 9F, Audio.
  19. Remove the mirror control switch.
  20. Remove the floor console. Refer to Section 9G, Floor Console.






  21. Remove the bolts and the screws from the instrument panel assembly.
  22. Remove the instrument panel from the vehicle.



Installation Procedure

  1. Install the instrument panel with the bolts and screws.

    2. Tighten
    Tighten the instrument panel bolts to 6 Nâ€¢m (53 lb-in).

  2. Install the floor console. Refer to Section 9G, Floor Console.
  3. Install the mirror control switch.
  4. Install the front speaker and speaker cover. Refer to Section 9F, Audio.
  5. Install the passenger's airbag module to the instrument panel. Refer to Section 8B, Supplemental Inflatable Restraints (SIR) (For vehicle equipped with the airbag).
  6. Install the glove box assembly to the instrument panel. Refer to "Glove Box"in this section.
  7. Install the hood latch release lever to the instrument panel. Refer to Section 9R, Body Front End.
  8. Install the audio system to the instrument panel. Refer to Section 9F, Audio System.
  9. Install the A/C controller to the instrument panel. Refer to Section 7B, Manual Control Heating, Ventilation, and Air Conditioning System.
  10. Install the instrument cluster assembly. Refer to "Instrument telltail"in this section.
  11. Install the front defroster grille. Refer to "Front Defroster Grille"in this section.
  12. Install the meter cluster. Refer to "Meter Cluster"in this section.
  13. Install the cluster molding to the instrument panel with the screws.
  14. Install the A-pillar trim panels. Refer to Section 9G, Interior Trim.
  15. Install the front rocker trim panels. Refer to Section 9G, Interior Trim.
  16. Install the signal lamp switch and the wiper switch. Refer to Section 6E, Steering Wheel and Column.
  17. Install the steering column trim cover. Refer to Section 6E, Steering Wheel and Column.
  18. Install the steering wheel to the steering column.
  19. Install the driver's airbag module to the steering wheel. Refer to Section 8B, Supplemental Inflatable Restraints (SIR) (For vehicle equipped with the airbag).
  20. Install the steering wheel cap.
  21. Connect the negative battery cable.

    3. Tighten
    Tighten the tie-bar bolts to 13 Nâ€¢m (115 lb-in)

  4. Instal the steering column with the nuts.
  5. Install the defroster duct hose with the screws.
  6. Instal the instrument panel assembly. Refer to "Instrument Panel"in this section.
  7. Connect the negative battery cable.
