Manual Transaxle Assembly

Tools Required

DW110-021 Engine Support Fixture

Removal Procedure

  1. Remove the air cleaner assembly. Refer to Section 1B, SOHC Engine Mechanical.
  2. Remove the battery and battery tray. Refer to Section 1E, Engine Electrical.
  3. Disconnect the select and the shift cable.
    • Remove the cable pins (1).
    • Remove the washers (2).
    • Disconnect the select and the shift cable (3).
    • Remove the cable E-rings (4).
    • Disconnect the cables from the cable bracket.



  4. Disconnect the engine wiring harness bending strap.
  5. Disconnect the ground wire, O2 sensor switch connector and the backup lamp switch connector.
    • Remove the ground wire bolt and disconnect the wire (1).
    • Disconnect the O2 sensor switch connector (2).
    • Disconnect the backup lamp switch connector (3).



  6. Disconnect the radiator lower hose.
    • Remove the bolts (1).
    • Disconnect the radiator lower hose (3).
  7. Remove crankshaft position (CKP) sensor.
    • Remove the bolt (2).
    • Disconnect the CKP sensor connector.
    • Remove the CKP sensor.
  8. Remove the transaxle under cover. Refer to Section 9N, Frame and Underbody.



  9. Remove the starter . Refer to Section 1E, Engine Electrical.



  10. Fix the engine assembly.
    • Remove the cowl panel weatherstrip.
    • Position the engine support fixture DW110-021 on the cowl panel and the front upper panel.
    • Tighten the engine fixture joint with a bolt after removing exhaust manifold bolt (No.4).



  11. Drain the transaxle fluid.
    • Remove the drain plug and drain the fluid.
  12. Disconnect the clutch cable.
    • Remove the cable adjust nut (1).
    • Disconnect the cable from the transaxle mount hole (2).



  13. Disconnect the vehicle speed sensor connector.



  14. Remove the front under longitudinal frames and stabilizer. Refer to Section 2C, Front Suspension.
  15. Remove the drive axle (only transaxle side). Refer to Section 3B, Manual Transaxle Drive Axle.



  16. Remove the clutch housing lower plate.
    • Remove the bolts (1).
    • Remove the lower plate (2).



  17. Remove the front exhaust pipe. Refer to Section 1G, Engine Exhaust.



  18. Remove the transaxle upper bolts.
    19. (1) Exhaust manifold side bolt.
    (2) Thermostat housing side bolt.



  19. Remove the reaction rod.
    • Reomove the bolts and nuts (1)(2).
    • Remove the reaction rod (3).



  20. Support the transaxle with a transaxle support jack.



  21. Remove the transaxle lower bolt and nut.
    22. (1) Lower bolt.
    (2) Lower nut.



  22. Remove the transaxle mounting bolts. Refer to "Transaxle Mount" in this section.



  23. Remove the manual transaxle assembly.
  24. Slide the transaxle sideways away from the engine block.
  25. Carefully lower the transaxle.



  26. Support the engine to normal position using the engine support fixture DW110-021.

    27. Notice : The abnormal position of the engine may damage to the related parts or interfere with them. You have to support the engine to normal position when removing the transaxle.

    Important : If it is impossible to use the special tool, support the engine to normal position with the auto jack.




Installation Procedure

  1. Support the transaxle using a support jack.
  2. Install the transaxle carefully.



  3. Install the transaxle lower bolt and nut.

    4. Tighten
    Tighten the bolt and nut to 55-65 N•m (41-48 lb-ft).

    (1) Lower bolt.
    (2) Lower nut.



  4. Install the transaxle mounting bolts. Refer to "Transaxle Mount" in this section.



  5. Install the reaction rod (3).

    6. Tighten
    Tighten the reaction rod bolt and nut (1) to 75-85 N•m (55-63 lb-ft).
    Tighten the reaction rod connection bolt and nut(2) to 75-85 N•m (55-63 lb-ft).




  6. Remove the support jack from the transaxle.



  7. Install the front exhaust pipe. Refer to Section 1G, Engine Exhaust.



  8. Install the transaxle upper bolts.

    9. Tighten
    Tighten the bolts to 55-65 N•m (41-48 lb-ft).

    (1) Exhaust manifold side bolt.
    (2) Thermostat housing side bolt.



  9. Install the clutch housing lower plate.



  10. Install the drive axle. Refer to Section 3B, Manual Transaxle Drive Axle.
  11. Install the front under longitudinal frames and stabilizer. Refer to Section 2C, Front Suspension.



  12. Remove the engine support fixture DW110-021.
    • Remove the engine support fixture from the cowl panel and the front upper panel.
    • Install the cowl panel weatherstrip.



  13. Install the starter. Refer to Section 1E, Engine Electrical.



  14. Connect the vehicle speed sensor connector.



  15. Install the radiator lower hose and crank position (CKP) sensor.

    16. Tighten
    Tighten the radiator lower hose bolt (1) to 8-15 N•m (70-132 lb-in).
    Tighten the CKP sensor bolt (2) to 5-8 N•m (44-70 lb-in).




  16. Connect the ground wire (1), the backup lamp switch connector (3) and O2 sensor switch connector (2).

    17. Tighten
    Tighten the ground wire bolt (1) to 15-22 N•m (11-16 lb-ft).

  17. Connect the engine wiring harness bending strap.



  18. Connect the select and shift cable.
    • Connect the select and shift cable (3) on the bracket.
    • Install the cable E-rings (4).
    • Install the washers (2).
    • Install the cable pins (1).
  19. Install the battery and battery tray. Refer to Section 1E, Engine Electrical.
  20. Install the air cleaner assembly. Refer to Section 1B, SOHC Engine Mechanical.



  21. Refill the transaxle fluid.

    22. Tighten
    Tighten the dram plug (a) to 25-30 N•m (18-22 lb-ft).

    • Remove the oil level plug (1).
    • Refill recommended fluid to the proper level.

    Tighten
    Tighten the oil level plug to 36-54 N•m (26-40 lb-ft).




  22. Adjust the clutch cable. Refer to Section 5C, Clutch.
    • Adjust the clutch cable by the clutch cable adjust nut.
  23. Install the transaxle under cover. Refer to Section 9N, Frame and Underbody.



Gear Shift Control Case Assembly

Removal Procedure

  1. Remove the air cleaner assembly. Refer to Section 1B, SOHC Engine Mechanical.
  2. Remove the battery.
  3. Disconnect the select and the shift control cable.
    • Remove the cable pins (1).
    • Remove the washers (2).
    • Disconnect the select and the shift control cable (3).
    • Remove the cable E-rings (4).
    • Disconnect the cables from the cable bracket.



  4. Remove the select lever.
    • Remove the bolts (1).
    • Remove the select lever (2).



  5. Remove the shift interlock bolt (a).

    6. Important : Certainly remove the shift interlock bolt. Otherwise, the gear shift control case can not be removed.




  6. Remove the gear shift control case assembly.
    • Remove the bolts (1).

    Important : Make sure the gear shift lever is in NEUTRAL.

      • Remove the gear shift control case assembly (2).
      • Remove the gasket (3).




  7. Remove the shift lever.
    • Position the gear shift control case assembly to the vice with a protector.
    • Remove the lever pin with a pin punch and a hammer (1).
    • Remove the shift lever (2).
    • Remove the boot (3).



  8. Remove the select/shift shaft.
    • Remove the shift guide bolt (1).
    • Remove the select/shift shaft (2).



  9. Remove the gear shift control case plate.
    • Remove the bolts (1).
    • Remove the Plate from the case (2).



Inspection Procedure

  1. Inspect for worn or damaged the fifth/reverse shift cam (1).
  2. Inspect for worn or damaged shift interlock plate (2).
  3. Inspect for worn or damaged select/shift lever (3).
  4. Inspect for bent or damaged select/shift shaft (4).



Installation Procedure

  1. Install the shift guide bolt.

    2. Tighten
    Tighten the shift guide bolt to 18-28 N•m (13-21 lb-ft).




  2. Install the new lever pin (1), boot (3) and shift lever (2).



  3. Install the gear shift control case and the select lever.

    4. Tighten

    • Tighten the gear shift control case bolts to 18-28 N•m (13-21 lb-ft).
    a. Gear shift control case bolt.
    • Tighten the select lever bolts to 18-28 N•m (13-21 lb-ft).
    b. Select lever bolt.

    Important : Make sure a correct bolt. There is length difference between select lever bolts.




  4. Install the shift interlock bolt.

    5. Tighten
    Tighten the shift interlock bolt to 18-28 N•m (13-21 lb-ft).




  5. Connect the select and shift cable.
    • Connect the select and shift cable (3) on the bracket.
    • Install the cable E-ring (4).
    • Install the washers (2).
    • Install the cable pins (1).
  6. Install the battery and battery tray. Refer to Section 1E, Engine Electrical.
  7. Install the air cleaner assembly. Refer to Section 1B, SOHC Engine Mechanicl.



Transaxle Mount

Removal Procedure

  1. Remove the air cleaner assembly. Refer to Section 1B, SOHC Engine Mechanical.
  2. Remove the battery and tray. Refer to Section 1E, Engine Electrical.



  3. Support the transaxle with a supporting jack.
  4. Remove the bolts and nuts from the bracket.
  5. Remove the bracket.



Installation Procedure

  1. Install the bracket, bolts and nuts.

    2. Tighten
    Tighten the mounting bracket bolts to 55-65 N•m (41-48 lb-ft).
    Tighten the mounting cage nuts to 55-65 N•m (41-48 lb-ft).




  2. Install the battery. Refer to Section 1E, Engine Electrical.
  3. Install the air cleaner assembly. Refer to Section 1B, SOHC Engine Mechanical.



Speedometer Driven Gear

(Left-Hand Drive Shown, Right-Hand Drive Similar)

Removal Procedure

  1. Disconnect the negative battery cable.
  2. Disconnect the vehicle speed sensor connector.
  3. Remove the bolt and the speedometer driven gear assembly.



  4. Separate the vehicle speed sensor from the speedometer driven gear.
    5. (1) Vehicle speed sensor.
    (2) Speedometer driven gear.

    Caution : Be careful to prevent personal injury while the exhaust pipe is hot.




Inspection Procedure

  1. Remove the O-ring from the speedometer driven gear housing.
  2. Remove the driven gear pin and disconnect the driven gear.
    • Check for a damaged or torn O-ring.
    • Check for a worn or damaged tooth of driven gear.
    (1) O-ring.
    (2) Driven gear pin.
    (3) Driven gear.



Installation Procedure

  1. Install the vehicle speed sensor to the speedometer driven gear.



  2. Install the speedometer driven gear assembly to transaxle housing.

    3. Tighten
    Tighten the speedometer driven gear assembly bolt to 4-7 N•m (35-62 lb-in).

  3. Connect the vehicle speed sensor connector.
  4. Connect the negative battery cable.



Gear Shift Control Cable

Removal Procedure

  1. Remove the air cleaner assembly. Refer to 1B, SOHC Engine Mechanical.
  2. Remove the battery.
  3. Disconnect the transaxle side select and shift control cable.
    • Remove the cable pins (1).
    • Remove the washers (2).
    • Disconnect the select and shift control cable (3).
    • Remove the cable E-rings (4).
    • Disconnect the cables from the cable bracket.



  4. Remove the floor console. Refer to Section 9G, Interior Trim.



  5. Disconnect the gear shift lever side select and shift control cable.
    • Remove the pins, E-rings and clips.



  6. Remove the select and shift control cable.
    • Remove the nuts.
    • Pull the cables out in the passenger room.



Installation Procedure

  1. Push the cables toward the engine compartment through dash panel's hole slightly.
  2. Position the cables on the select and the shift lever.



  3. Install the select and shift cable nut and cable grommet.

    4. Tighten
    Tighten the nut to 4-7 N•m (35-62 lb-in).




  4. Connect the gear shift lever side select and shift control cable.
    • Install the pins, E-rings and clips.



  5. Connect the select and shift cable.
    • Connect the select and shift cable (3) on the bracket.
    • Install the cable E-rings (4).
    • Install the washers (2).
    • Install the cable pins (1).



  6. Install the floor console. Refer to Section 9G, Interior Trim.
  7. Install the battery. Refer to Section 1E, Engine Electrical.
  8. Install the air cleaner assembly. Refer to Section 1B, SOHC Engine Mechanical.



Gear Shift Control Lever

Removal Procedure

  1. Remove the floor console. Refer to Section 9G, Interior Trim.
  2. Disconnect the select and shift control cable. Refer to "Gear Shift Control Cable" in this section.
  3. Remove the gear shift control lever assembly.
    • Remove the bolts (1).
    • Remove the gear shift control lever assembly (2).



Installation Procedure

  1. Install the gear shift control lever assembly.

    2. Tighten
    Tighten the bolts to 4-7 N•m (35-62 lb-in).

  2. Connect the select and shift control cable. Refer to "Gear Shift Control Cable" in this section.
  3. Install the floor console. Refer to Section 9G, Interior Trim.
