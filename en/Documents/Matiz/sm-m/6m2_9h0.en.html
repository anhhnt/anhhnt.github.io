MAINTENANCE AND REPAIR

ON-VEHICLE SERVICE




Front Seats

Removal Procedure

  1. Remove the bolts that secure the rear portion of the seat to the floor.
  2. Remove the bolts that secure the front portion of the front seat to the floor.
  3. Disconnect the electrical connector from the seat.
  4. Remove the seat.



Installation Procedure

  1. Install the seat.
  2. Connect the electrical connector to the seat.

    3. Notice : Dissimilar metals in direct contact with each other may corrode rapidly. Make sure to use the correct fasteners to prevent premature corrosion.

  3. Install the bolts into the front portion of the front seat.

    4. Tighten
    Tighten the front seat bolts to 4 N•m (3 lb-ft).

  4. Install the bolts into the rear portion of the front seat.

    5. Tighten
    Tighten the front seat bolts to 25 N•m (18 lb-ft).




Head Restraint

Removal Procedure

  1. Press the head restraint adjust button and remove the head restraint from the seatback.



Installation Procedure

  1. Install the head restraint into the guide sleeves. Press down in order to engage the stop pin.



Front Seatback

Removal Procedure

  1. Disconnect the negative battery.
  2. Remove the front seat assembly. Refer to "Front Seats" in this section.
  3. Remove the recliner knob from the front seat.



  4. Remove the recliner cover from the front seat.



  5. Remove the seatback cover.
  6. Remove the recliner from the front seat.



  7. Remove the seatback bolt from the seat cushion.
  8. Remove the seatback assembly.



Installation Procedure

  1. Install the seatback assembly with the bolt.
  2. Install the recliner with the bolts.

    3. Tighten
    Tighten the recliner bolts to 24 N•m (17 lb-ft).

  3. Install the seatback cover with the hog rings.



  4. Install the recliner knob and the cover with the screws.
  5. Install the front seat assembly. Refer to "Front Seats" in this section.
  6. Connect the negative battery cable.



Front Seat Cushion

Removal Procedure

  1. Disconnect the negative battery cable.
  2. Remove the front seat assembly. Refer to "Front Seat" in this section.
  3. Remove the seat belt buckle. Refer to Section 8A, Seat Belts.
  4. Remove the recliner knob.



  5. Remove the recliner cover.



  6. Remove the seat cushion.



Installation Procedure

  1. Install the seat cushion with the seatback bolt and the recliner bolts.

    2. Tighten
    Tighten the seat cushion bolts to 24 N•m (17 lb-ft).




  2. Install the recliner cover and the recliner knob with the screws.
  3. Install the seat belt buckle. Refer to Section 8A, Seat Belts.
  4. Install the front seat assembly. Refer to "Front Seats" in this section.
  5. Connect the negative battery cable.



Front Seats Guide Rail

Removal Procedure

  1. Disconnect the negative battery cable.
  2. Remove the front seat assembly. Refer to "Front Seats" in this section.
  3. Remove the guide rail from the seat.



Installation Procedure

  1. Install the guide rail to the seat with the bolts.

    2. Tighten
    Tighten the guide rail bolts to 24 N•m (17 lb-ft).

  2. Install the front seat assembly. Refer to "Front Seat" in this section.
  3. Connect the negative battery cable.



Front Seats Cushion Frame

Removal Procedure

  1. Disconnect the negative battery cable.
  2. Remove the front seat cushion. Refer to "Front Seat Cushion" in this section.
  3. Remove the bolts and the guide rail.



  4. Remove the seat cushion cover.
  5. Remove the seat cushion frame.



Installation Procedure

  1. Install the seat cushion cover with the hog rings.



  2. Install the guide rail with the bolts.
  3. Install the front seat cushion. Refer to "Front Seat Cushion" in this section.
  4. Connect the negative battery cable.



Rear Seatback

Removal Procedure

  1. Remove the rear seatback lock by pulling the lock knob upward.
  2. Remove the seatback hinge bolts and nut.
  3. Remove the rear seatback.



Installation Procedure

  1. Install the rear seatback with the hinge bolts and nut.

    2. Tighten
    Tighten the hinge bolts to 24 N•m (17 lb-ft).




Rear Seat Cushion

Removal Procedure

  1. Press the hinge wire.
  2. Remove the rear seat cushion.



Installation Procedure

  1. Press the hinge wire and set to the rear seat cushion.
  2. Install the rear seat cushion



Rear Seatback Latch

Removal Procedure

  1. Remove the rear seatback assembly. Refer to "Rear Seatback" in this section.
  2. Remove the lock knob from the seatback.



  3. Remove the rear seatback cover from the seatback frame.
  4. Remove the lock from the seatback frame.



Installation Procedure

  1. Install the lock to the seatback frame with the bolts.
  2. Install the rear seatback cover to the seatback frame.



  3. Install the lock knob to the seatback.
  4. Install the rear seatback assembly. Refer to "Rear Seatback" in this section.

GENERAL DESCRIPTION AND SYSTEM OPERATION

Seats

Important : Do not attempt to change the designed seat position by altering the designed seat adjuster-to-floor pan anchor provisions or the seat adjuster-to-seat frame anchor provisions. Changing the seat position could affect the performance of the seat system.

This vehicle is equipped with low-backed front seats with separate head restraints and a three-passenger folding rear seat. Split folding rear seat is optional. Seats cushions and seatbacks have formed foam pads, which fit the contours of the full-panel seatback frame assembly and the designed contour of the seat cushion frame. There are no front seat forward or rearward relocation provisions provided at either the seat adjuster-to-seat frame or the seat adjuster-to-floor pan anchor attachments.
