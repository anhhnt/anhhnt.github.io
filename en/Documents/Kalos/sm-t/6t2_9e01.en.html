MAINTENANCE AND REPAIR

ON-VEHICLE SERVICE




Cigar Lighter

Removal Procedure

  1. Disconnect the negative battery cable.
  2. Remove the cigar lighter from the cigar lighter housing.
  3. Remove the floor console. Refer to Section 9G, Floor Console
  4. Remove the cigar lighter housing from the instrument Panel.
  5. Disconnect the cigar lighter electrical connector.



Installation Procedure

  1. Connect the cigar lighter electrical connector.
  2. Install the cigar lighter housing in the instrument panel.
  3. Install the floor console, Refer to Section 9G, Floor Console
  4. Install the cigar lighter in the cigar lighter housing.
  5. Connect the negative battery cable.



Ashtray

Removal Procedure

  1. Remove the ashtray.



Installation Procedure

  1. Install the ashtray.



Cupholder

Removal Procedure

  1. Remove the cupholder from the instrument panel.



Installation Procedure

Glove Box

Installation Procedure

Digital Clock

Removal Procedure

  1. Disconnect the negative battery cable.
  2. Remove the instrument panel center upper cover.
  3. Remove the screws and the clock.



Installation Procedure

  1. Install the clock with the screws.

    2. Tighten
    Tighten the clock screws to 3 N•m (27 lb-in).

  2. Install the instrument panel center upper cover.
  3. Connect the negative battery cable.



Instrument Cluster

Removal Procedure

  1. Disconnect the negative battery cable.
  2. Remove the instrument cluster front trim panel.



  3. Remove the instrument cluster rear trim panel.



  4. Remove the screws and the instrument cluster.
  5. Disconnect the electrical connectors.



Installation Procedure

  1. Connect the electrical connectors.
  2. Install the instrument cluster with the screws.



  3. Install the instrument cluster rear trim panel.



  4. Install the instrument cluster trim panel.
  5. Connect the negative battery cable.



Instrument Cluster Indicator Lamps

Removal Procedure

  1. Disconnect the negative battery cable.
  2. Remove the instrument cluster. Refer to " Instrument Cluster " in this section.

Installation Procedure

  1. Install the new bulb.
  2. Install the instrument cluster. Refer to " Instrument Cluster " in this section.

Instrument Panel

Removal Procedure

  1. Disconnect the negative battery cable.
  2. Remove the driver's airbag module from the steering wheel. Refer to Section 8B, Supplemental Inflatable Restraints (SIR) (For the vehicle equipped with the airbag.)
  3. Remove the steering wheel from the steering column. Refer to Section 6E, Steering Wheel and Column.
  4. Remove the steering column trim cover. Refer to Section 6E, Steering Wheel and Column.

  5. Remove the signal lamp switch and the wiper switch. Refer to Section 6E, Steering Wheel and Column.
  6. Remove the A-pillar trim panels. Refer to Section 9G, Interior Trim.
  7. Remove the instrument cluster assembly. Refer to "Instrument cluster" in this section.
  8. Remove the side cover and knee bolster.
  9. Remove the center vents.
  10. Remove the cigar lighter and ashtray.
  11. Remove the cupholder.



  12. Remove the A/C controller from the instrument. Refer to Section 7B, Manual Control Heating, Ventilation, and Air Conditioning System(HVAC).
  13. Remove the audio system from the instrument panel. Refer to Section 9F, Audio Systems.
  14. Remove the digital clock.
  15. Remove the glove box assembly from the instrument panel. Refer to "Glove Box" in this section.
  16. Disconnect the ALDL connector.



  17. Remove the instrument panel bolts behind the heating, ventilation, and air conditioning (HVAC) controls.
  18. Remove the instrument panel bolts above the steering column.
  19. Remove the instrument panel bolts from the ends of the instrument panel.



  20. Remove the instrument panel end screws.
  21. Remove the tie-bar retaining bolts.
  22. Remove the instrument panel from the vehicle.



Installation Procedure

  1. Install the instrument panel assembly with the bolts and screws.

    2. Tighten
    Tighten the instrument panel bolts to 20 N•m (15 lb-ft).

  2. Connect ALDL connector with the screws.
  3. Install glove box assembly from the instrument panel. Refer to "Glove Box" in this section.



  4. Install the digital clock.
  5. Install the audio system from the instrument panel. Refer to Section 9F, Audio Systems.
  6. Install the A/C controller from the instrument. Refer to Section 7B, Manual Control Heating, Ventilation, and Air Conditioning (HVAC)System.

    7. Tighten
    Tighten the instrument panel bolts behind the HVAC controls to 4 N•m (35 lb-in).

  7. Install the cupholder.
  8. Install the cigar lighter and ashtray.

  9. Install the clock.
  10. Install the knee bolster and the instrument panel lower cover.
  11. Install the side cover.
  12. Install the instrument cluster assembly. Refer to "Instrument cluster" in this section.
  13. Install the A-pillar trim panels. Refer to Section 9G, Interior Trim.
  14. Install the signal lamp switch and the wiper switch. Refer to Section 6E, Steering Wheel and Column.
  15. Install the steering column trim cover. Refer to Section 6E, Steering Wheel and Column.
  16. Install the steering wheel to the steering column. Refer to Section 6E, Steering Wheel and Column.
  17. Install the driver's airbag module from the steering wheel. Refer to Section 8B, Supplemental Inflatable Restraints(SIR) (For vehicle equipped with the airbag).
  18. Connect the negative battery cable.
